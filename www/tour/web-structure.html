Unison Tour: Viewing Structures

Structural Context of Sequence Features
http://unison-db.org/pseq_structure.pl?q=TNFA_HUMAN;userfeatures=Estrand@164-174,mysnp@170
The "Structure" tab shows structures associated with the query sequence, including conservative comparative modelling structures. Please be sure to click the TNF model and a SNP to see the on-the-fly highlighting. Currently, we display only SNPs from Swiss-Prot. Users may specify their own regions to highlight using the userfeatures URL parameter, as shown above. Within Genentech, this tab also displays homology models using protein threading, but these data are not part of the public release.
