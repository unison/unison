The Unison Tour is being overhauled. The following links provide a few examples of how Unison may be used. We expect the new Unison Tour pages to be available in mid-September.
Sequence Analysis (i.e., show information for a given sequence)
http://unison-db.org/cgi/pseq_summary.pl?q=TNFA_HUMAN
The "Summary" tab provides an overview for a sequence. The second row of tabs navigate across other details for this sequence.
Sequence Features
http://unison-db.org/cgi/pseq_features.pl?q=TNFA_HUMAN
The "Features" tab graphically displays many additional features which are not shown on the "Summary" tab.
Structural Context of Sequence Features
http://unison-db.org/cgi/pseq_structure.pl?q=TNFA_HUMAN;userfeatures=Estrand@164-174,mysnp@170
The "Structure" tab shows structures associated with the query sequence, including conservative comparative modelling structures. Please be sure to click the TNF model and a SNP to see the on-the-fly highlighting. Currently, we display only SNPs from Swiss-Prot. Users may specify their own regions to highlight using the userfeatures URL parameter, as shown above. Within Genentech, this tab also displays homology models using protein threading, but these data are not part of the public release.
Mining for Sequences Using Predifined Criteria
http://unison-db.org/cgi/browse_views.pl?cv_id=5
Frequently used queries are stored as "canned views". This makes it easy to bookmark dynamic queries, and this is the primary mechanism I use to convey search results to others. In a near-future release, we will permit restriction by species and sequence "age".
Mining for Sequences Using User Criteria
http://unison-db.org/cgi/search_sets.pl
This page integrates precomputed results from HMMs, PSSMs, and Prospect threading for family-specific mining. The left side allows the user to define sequence selection criteria. At the top is menu of curated sequence sets. In the default example, sequences are selected using models (i.e., HMMs, PSSMs, or PDBs) for TNF superfamily members, and "hits" are categorized according to the specified curated set of sequences.

We anticipate that the overhauled Unison Tour pages will include examples of conntecting to Unison using an interactive SQL interpreter, with Perl DBI, and with the Unison Perl API.
